verify_distinct(list_of_nodes)¶

Check that objects in list_of_nodes represent distinct (qu)bits.

list_of_nodes is a list containing nodes of type id, indexed_id, primary_list, or id_list. We assume these are all the same type ‘qreg’ or ‘creg’. This method raises an exception if list_of_nodes refers to the same object more than once.
