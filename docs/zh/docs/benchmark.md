###2.特殊基准测试电路
	• 高度并行化电路
        • 不同量子算法的结构允许不同程度的并行化，通过比较量子门数量，门数和电路深度的比率高度并行的应用将大量运算放入相对较小的电路深度中。
	• 高度串行化电路
	    • 定义：设置电路深度的最长路径上两个量子位相互作用的数量接近总的双比特数量。
	• 高度纠缠电路
	    • 定义：通过计算两个量子位相互作用的所有门操作的比例，测试电路种两个量子位相互作用程度。
	• 中间态测量电路
	    • 定义：对于多个连续层的门操作组成的电路，测量门在不同层数为程序执行期间和之后提取信息
