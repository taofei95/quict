# 模拟基准测试

## 状态向量模拟器基准测试

由5~25量子比特、50~250量子门数、随机指令集构造的的量子电路，测试用状态向量模拟器模拟量子电路运行过程中量子比特的状态时的运行时间，以测试结果作为模拟速度的基准。
